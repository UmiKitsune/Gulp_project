Git

  1. $git init - инициализируем git в проект
  2. $git config --global user.name "имя пользователя"
    $git config --global user.email ваш мейл@gmail.com
  3. $git add . - добавляем все в отслеживание изменений
  4. $git commit -m "комментарий измений" - комменитируем изменения
  5. $git status - статус изменений

Git Branches

  1. $git branch имя ветки - создаем новую ветку в проекте
  2. $git branch - посмотерть все ветки в проекте
  3. $git checkout имя ветки - переход в нужную ветку

Не забывайте использовать add и commit для "сохраниения" изменений!

Сливаем все ветки в основную ветку master

  1. $git merge имя ветки - собираем ветки в основной
  2. $git branch -D имя ветки - удаляем ненужные ветки

Gulp

  1. Необходимо установить Node.js и npm! Проверка что они есть: node -v и npm -v
  2. $npm i gulp-cli -g - устанавливаем gulp-cli глобально
  3. $npm init --yes - инициализация
  4. $npm i gulp - устанавливаем gulp локально в проект
  5. Cоздаем файлы.gitignore и gulpfile.js
  6. В.gitignore записываем файлы, которые не нужно отслежтвать.
    Например: node_modules.
  7. Cоздаем таски в gulpfile.js
  8. Дополнительно: $npm install browser-sync - для создания сервера

Sass

  1. $npm i gulp-sass - устанавливаем gulp в проект
  2. Создаем основной sass-файл и несколько мелких(если необходимо)
  3. Описываем task в gulpfile.js
  4. Прописываем стили

GitHub

  1. Регестрируемся на GitHub и создаем новый проект.
  2. $git remote add origin https://github.com/ваш логин/имя репозитория.git - связываем удаленный репозиторий с проектом.
  3. $git push -u origin master - запушиваем код.
  4. $git push - после этого пользуемся сокращенной записью.
